# 🤖 介绍
欢迎使用 AgentGPT 👋，这是一个自主 AI 代理平台，可以使用户在网页上创建和部署可定制的自主 AI 代理。
只需为您的 AI 代理指定名称和目标，就可以看到它开始了一段激动人心的旅程，以完成分配的任务。

## 它如何工作？
AgentGPT 通过链接语言模型（代理）来执行给定的目标。
递归地，代理会思考执行目标的最佳任务，执行这些任务，评估其表现，并不断思考更多任务。
