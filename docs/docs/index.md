# 🤖 项目介绍
欢迎使用 凹凸GPT 👋，这是一个由AgentGPT 分支出来的中文 AI 代理平台，可以使用户在网页上创建和部署可定制的 AI 代理。
只需为您的 AI 代理指定名称和目标，就可以看到它开始了一段激动人心的旅程，以完成指定任务。

## 它如何工作？
凹凸GPT  通过链接语言模型（代理）来执行给定的目标。通过链式思考执行目标任务，并评估其表现，并不断思考更多任务。
